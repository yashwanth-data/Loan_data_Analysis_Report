# Loan_Data_Analysis_Report
##EXECUTIVE SUMMARY: LOAN PORTFOLIO ANALYSIS
This report analyzes a dataset of 367 loan applications across 3 key regions in Bangalore. The analysis focuses on understanding applicant profiles and income levels to optimize loan distribution strategies.

##Key Insights:

***Market Concentration: KR Puram is the most active region with 138 applications, showing a healthy geographic spread across the Bangalore South district.

***Educational Profile: The dataset includes 283 Graduates and 84 Non-Graduates, suggesting a strong correlation between higher education and loan eligibility.

***Employment Impact: Self-employed applicants request higher loan amounts on average compared to salaried individuals, identifying a key segment for high-ticket loan products.

##Conclusion:
The portfolio is well-diversified. Future growth should focus on capturing the high-value self-employed segment while maintaining the stable volume provided by the Graduate demographic.
